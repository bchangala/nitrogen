NITROGEN
========

Welcome to the NITROGEN documentation website. The latest development
code is available at the `project repository <https://github.com/bchangala/nitrogen>`_ on GitHub.
Stable releases can be installed directly from the 
`Python Package Index <https://pypi.org/project/nitrogen/>`_::

    pip install nitrogen
